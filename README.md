## An easy but complete online infomation publish platform

1. running environment:
> Tomcat 7.0; JDK 1.7; SQL Server2008;

2. Methord used:
> Java Servlet; jstl;

3. structure:

* USER
```
  user signup/logout;
  admin add/delete/modify informaton;
  anonymous look through all the infomation;
```

* Part
```
  Machine Info;
  Books Info;
  Help Info;
  Service Info;
```
